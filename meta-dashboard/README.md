# FBWatson

FBWatson is an AI-powered analytics dashboard for Facebook Ads, designed to provide deep insights and clarity into your ad performance.

## Features

- **Multi-language Support**: English, Arabic, Hebrew, French, German.
- **AI Investigator**: Ask questions about your data in natural language.
- **Deep Insights**: Automated analysis of trends, creative performance, and ad fatigue.
- **Campaign Management**: View and filter campaigns, ad sets, and ads.
- **Custom Reports**: Flexible reporting with export options to Excel and Google Sheets.

## Getting Started
